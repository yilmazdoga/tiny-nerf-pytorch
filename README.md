# tint-nerf-pytorch
This project is a simplified and cropped down PyTorch implementation of NeRF.

## Installation
```
git clone https://github.com/yilmazdoga/tiny-nerf-pytorch.git
cd tiny-nerf-pytorch
pip install -r requirements.txt
```
